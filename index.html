Summary: Latitude vs. X

Weather data was extracted from the OpenWeatherMap API. This purpose of this project is to evaluate the temperature changes of a number of cities as we get closer to the equater.

The dataset includes 500 cities pulled from the API. The data was pulled from the API into jupyter notebook utilizing pandas to complete an analysis of the dataset. Four factors were evaluated against latitude: temperature, cloudiness, windspeed, and humidity.

This site provides the various visualizations created as part of the anlysis, as well as explanations and descriptions of any trends and correlations witnessed.
